## tvN 나의 영어 사춘기 과제를 공부하기 위한 사이트

## Folder Structure

After creation, your project should look like this:

```
my-app/
  README.md
  node_modules/
  package.json
  public/
    index.html
    homework1.csv
    homework2.csv
    homework3.csv
    favicon.ico
  src/
    client/
      Root.js
    components/
      EnList.js
      EnListItem.js
      Home.js
      Navbar.js  
    App.css
    App.js
```

## Development

Nodejs 및 Npm 설치 후,
```
# Install Packages
npm install
# Run Dev
npm start
    
```
